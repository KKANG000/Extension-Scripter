# 🎬 Extension-Scripter

**SillyTavern을 위한 강력한 서사 제어 및 장면 관리 도구**

Extension-Scripter는 당신을 단순한 채팅 참여자에서 **'총괄 감독(Director)'**으로 변모시킵니다.
AI를 **'수석 작가(Lead Author)'**로 설정하여, 장면의 시작과 끝, 전개 속도, 그리고 앞으로 일어날 사건의 흐름을 완벽하게 통제하세요.

## ✨ 주요 기능

### 1. 3가지 서사 프로토콜 (Narrative Protocols)
AI의 집필 스타일을 실시간으로 지휘할 수 있습니다.
*   **🟢 READY (Scene Start)**: 새로운 장면의 시작을 알립니다. 이전 맥락을 잇되, 새로운 분위기로 환기시킵니다.
*   **🔴 ACTION (Writing in Progress)**: 장면을 깊이 있게 묘사합니다. 사건을 급하게 끝내지 않고, 감정선과 디테일을 파고들도록(Micro-progression) 유도합니다.
*   **🔵 MONTAGE (Fast Pace)**: 시간과 공간을 압축하여 빠르게 전개합니다. 지루한 이동 과정이나 긴 시간을 몽타주 기법으로 요약할 때 유용합니다.

### 2. 완벽한 장면 관리 (Scene Management)
*   **Current Scene**: 현재 진행 중인 장면의 지시문만 입력하세요.
*   **Next Scene**: 다음에 일어날 사건(스포일러)을 미리 적어두세요. AI는 이 내용을 알지만, **절대 미리 발설하지 않고** 현재 장면에만 집중합니다.
*   **Scene Queue**: 떠오르는 아이디어를 대기열에 미리 적어두고 관리하세요.
*   **🎬 CUT!**: 현재 장면이 만족스럽게 끝났나요? 'CUT!' 버튼을 누르면 즉시 다음 장면(Next Scene)을 가져오고, 새로운 씬을 시작할 준비를 마칩니다.

### 3. 감독의 개입 (Director's Note)
*   **Core Principle**: 이야기 전체를 관통하는 철학이나 절대적인 규칙을 주입할 수 있습니다.
*   **Caution**: AI가 피해야 할 표현이나 주의사항을 실시간으로 전달하세요.
*   **Final Check**: AI가 답변을 내놓기 직전, 스스로 개연성과 설정 오류를 검토하도록 강제합니다.

### 4. 편리한 인터페이스
*   **사이드바 & 팝업**: 채팅창 옆에서 바로 시나리오를 수정하거나, 넓은 화면으로 작업할 수 있습니다.
*   **Glassmorphism 테마**: 세련된 반투명 디자인을 지원합니다.
*   **단축키 지원**: `Ctrl + Shift + C`로 빠르게 'CUT!'을 외치세요.

---

## 📦 설치 방법

1. SillyTavern의 `extensions/third-party` 폴더 안에 `Extension-Scripter` 폴더를 위치시킵니다.
2. SillyTavern을 재시작하거나 브라우저를 새로고침합니다.
3. 상단 확장 프로그램 메뉴(마법봉 아이콘) 또는 채팅창 입력칸 옆의 **슬레이트(🎬) 아이콘**을 클릭하여 활성화합니다.

## 🚀 사용 가이드 (Quick Start)

1. **시나리오 작성**: 사이드바를 열고 `Current Scene`에 지금 보고 싶은 장면을 적습니다. (예: "두 사람이 빗속에서 처음 마주친다.")
2. **레디, 액션!**: 상태를 `READY`로 두고 메시지를 보냅니다. AI가 장면의 오프닝을 엽니다.
3. **디테일 묘사**: AI의 응답이 마음에 들면, 상태가 자동으로(설정 시) 또는 수동으로 `ACTION`으로 변경됩니다. 이제 AI는 그 장면을 밀도 있게 묘사합니다.
4. **장면 전환**: 장면이 절정에 달했거나 마무리하고 싶다면, `Next Scene`에 다음 내용을 적거나 대기열에서 불러온 뒤 **[CUT!]** 버튼을 누르세요.

## ⚠️ 주의사항
*   이 확장 프로그램은 SillyTavern의 최신 버전을 권장합니다.
*   AI 모델의 성능에 따라 지시 이행 능력에 차이가 있을 수 있습니다.
